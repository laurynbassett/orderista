# Orderista

_Restaurant Map Finder_

This website was built over 3 days during my review week at Fullstack Academy.

## Tech Stack

I used the following technologies to build this app:

- React
- Redux
- Express
- Sequelize
- PostgreSQL
- Zomato API
- Mapbox GL JS
- Material-UI
